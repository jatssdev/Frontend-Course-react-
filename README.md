# 🚀 Front‑End Development Course

**From Zero to Production‑Ready**

> **Offered by VJ. NJ. MJ. Private ITI**

Learn to build fast, accessible, and beautiful web apps with **HTML • CSS • JavaScript • React • Tailwind CSS**.

---

## 1. Why This Course?

* **Hands‑on pedagogy** – every concept is reinforced through guided labs and mini‑projects.
* **Beginner‑friendly → Professional‑ready** – start from first principles, finish with production techniques.
* **Modern tooling** – Vite, VS Code, Git/GitHub, browser DevTools, ESLint & Prettier.
* **Career focus** – finish with a portfolio and interview‑ready skills.

## 2. Who Should Enroll?

Anyone eager to become a professional front‑end developer or upskill for modern web projects. **No prior coding experience required** – just curiosity and commitment.

---

## 3. Course Roadmap & Syllabus

Below is a **deep‑dive, week‑by‑week plan** detailing concepts, labs, and deliverables.

| Week      | Module                             | Key Topics                                                                                                                                                          | Lab / Assignment                                                           |
| --------- | ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| **1**     | **HTML Foundations**               | • Document structure & tags  <br>• Block vs. inline  <br>• Head elements & metadata  <br>• Lists, tables, images, iframes                                           | **Lab 1:** Build a personal profile page                                   |
| **2**     | **Semantic HTML5 & Accessibility** | • Sectioning elements (header, nav, main, aside, footer)  <br>• Semantic text (article, figure, time)  <br>• Forms & native validation  <br>• ARIA roles, landmarks | **Lab 2:** Form‑driven “Contact Us” page with semantic markup & a11y audit |
| **3**     | **CSS Core Concepts**              | • Selectors & specificity  <br>• Box model, display types  <br>• Units (px, em, rem, vw/vh)  <br>• Float vs. Flexbox intro                                          | **Lab 3:** Clone a popular blog layout using float + clearfix              |
| **4**     | **Layout Mastery**                 | • Flexbox deep dive  <br>• CSS Grid two‑dimensional layout  <br>• Responsive design, mobile‑first  <br>• Media queries & container queries                          | **Lab 4:** Build a responsive dashboard grid that collapses gracefully     |
| **5**     | **Modern CSS Techniques**          | • Custom properties (CSS vars)  <br>• Transforms, transitions, keyframe animations  <br>• Gradient backgrounds & masks  <br>• CSS architecture (BEM, OOCSS)         | **Lab 5:** Animated product card set with hover effects                    |
| **6**     | **JavaScript Fundamentals**        | • Primitives & objects   <br>• Control flow  <br>• Functions & scope  <br>• DOM selection & events                                                                  | **Quiz 1** + **Lab 6:** Interactive to‑do list (CRUD in vanilla JS)        |
| **7**     | **Modern JS (ES6+)**               | • let/const, arrow functions, template literals  <br>• Destructuring & spread  <br>• Modules & bundling with Vite  <br>• Fetch API & Promises                       | **Lab 7:** Consume a public REST API & render cards dynamically            |
| **8**     | **Asynchronous JS & Tooling**      | • async/await patterns  <br>• Error handling  <br>• NPM scripts & package.json  <br>• Linting with ESLint, Prettier                                                 | **Lab 8:** Build a weather app with loading states & error boundaries      |
| **9**     | **JavaScript Testing**             | • Introduction to unit testing  <br>• Jest & Vitest basics  <br>• Testing DOM with @testing-library  <br>• Coverage reports                                         | **Lab 9:** Write unit tests for weather app utilities                      |
| **10**    | **React Basics**                   | • Why React? Virtual DOM  <br>• JSX syntax  <br>• Functional components, props  <br>• useState & basic state patterns                                               | **Lab 10:** Counter & color‑picker components                              |
| **11**    | **React Hooks & Routing**          | • useEffect lifecycle  <br>• Derived & reducer state  <br>• React Router v6  <br>• Nested routes & params                                                           | **Lab 11:** Multi‑page blog with dynamic post route                        |
| **12**    | **State Management**               | • Context API  <br>• Custom hooks  <br>• Performance (memo, useCallback)  <br>• Intro to Redux Toolkit                                                              | **Lab 12:** Theme switcher with persisted dark‑mode                        |
| **13**    | **Forms & Validation in React**    | • Controlled vs. uncontrolled  <br>• Formik & React‑Hook‑Form  <br>• Yup validation  <br>• Error UX                                                                 | **Lab 13:** Checkout form with live validation                             |
| **14**    | **React Performance & Patterns**   | • Code‑splitting with lazy & Suspense  <br>• React 18 concurrency features  <br>• Compound components  <br>• Render props & custom hook patterns                    | **Quiz 2** + **Lab 14:** Progressive image gallery with lazy loading       |
| **15**    | **Tailwind CSS Fundamentals**      | • Utility‑first philosophy  <br>• Config file & custom theme  <br>• Responsive, pseudo, dark mode  <br>• @apply & component extraction                              | **Lab 15:** Rebuild Lab 5 UI in Tailwind                                   |
| **16**    | **Tailwind Advanced Techniques**   | • Plugin ecosystem  <br>• Animation utilities  <br>• JIT compiler & purging  <br>• Integrating with React/Next.js + Headless UI                                     | **Lab 16:** Dashboard skeleton loader & modal component                    |
| **17‑18** | **Capstone Project Sprint**        | End‑to‑end SPA using React + Tailwind. Students plan, prototype in Figma, implement with Vite, test, and deploy to Netlify/Vercel.                                  | **Deliverable:** Production‑ready project with README & live URL           |

*Total classroom time: **108 hours** · Estimated self‑study: **50 hours***
